# Campus-Life-Super-App

The Campus Life Super App is a multi-page web application designed to help students manage daily challenges like tracking assignments, locating parking areas, and finding study spots on campus. The goal of the application is to have a one stop shop that is simple and user-friendly.

The application was built to improve organization.

I used HTML, Bootstrap, javaScript, LocalStorage, and Open-Meteo API

There is a home page, an assignments page, a campus map page,and a request page. 

To use, clone the repository and open index.html in a browser. You can navigate the pages using the nav bar.

For future improvements I would want to make it prettier and to add a place for the assignments to actually show up and for requests to be sent.
